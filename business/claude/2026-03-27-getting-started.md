# Business Notes — Getting Started

This note initializes the `business/` section of the repository. Add work-related notes generated by AI assistants here.

## Suggested Sub-topics

Use tags and subfolders to organize notes by theme. Common business note topics include:

- **Strategy** — plans, goals, OKRs
- **Projects** — meeting summaries, action items, status updates
- **Clients** — research, communication logs, context
- **Finance** — budgeting, forecasting, analysis
- **Processes** — SOPs, workflows, documentation

## File Naming Convention

Follow the same convention as all notes in this repository:

```
YYYY-MM-DD-descriptive-title.md
```

Example: `2026-04-01-q2-strategy-review.md`
