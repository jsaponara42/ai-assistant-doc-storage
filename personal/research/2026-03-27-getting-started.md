# Getting Started with AI Assistant Doc Storage

This note serves as a reference example for the format all notes in this repository should follow. It lives in `personal/research/` and uses frontmatter to track that it was generated by Claude.

## What This Repository Is For

This repository stores markdown notes created by AI assistants (Claude, ChatGPT, Gemini, etc.). Notes are organized by context (`personal` vs. `business`) and note type (`projects`, `research`, `ideas`).

## How to Navigate

- Browse `personal/` for notes on personal topics.
- Browse `business/` for notes on work-related topics.
- Use frontmatter (especially `ai`, `tags`, and `summary`) to route and retrieve notes quickly.

## Tips for Getting the Most Out of These Notes

1. **Use tags** to find related notes across different AI assistants or contexts.
2. **Link related notes** using Obsidian wiki-links: `[[note-filename]]`.
3. **Review the `summary` frontmatter field** for a quick overview before reading the full note.
